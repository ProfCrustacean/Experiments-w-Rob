# Module Card: learning-apply

Owner: self-improvement-flow-owner

## Purpose

Apply allowed proposals while enforcing harness and safety policy.

## When To Use

When gate policy permits full or partial apply.

## Inputs

Proposals, apply policy mode, caps and thresholds.

## Outputs

Applied changes and apply summary details.

## Steps

1. Filter by policy.
2. Apply change set.
3. Persist applied changes.

## Failure Signals

Apply blocked by policy or patch write failure.
